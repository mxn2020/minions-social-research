# Welcome to Minions Social-research

We're excited to introduce **Minions Social-research** — Trend sources, competitor snapshots, hashtag sets, and audience notes.

## What's Included

- **TypeScript SDK**: `npm install @minions-social-research/sdk`
- **Python SDK**: `pip install minions-social-research`
- **CLI**: `npm install -g @minions-social-research/cli`

## Getting Started

Check out the [documentation](https://social-research.minions.help) for a full guide.
