---
title: Installation
description: Install Minions Social-research packages
---

## TypeScript / Node.js

```bash
npm install @minions-social-research/sdk minions-sdk
```

## Python

```bash
pip install minions-social-research
```

## CLI

```bash
npm install -g @minions-social-research/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
