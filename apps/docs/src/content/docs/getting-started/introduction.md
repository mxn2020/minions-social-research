# Minions Social-research

Trend sources, competitor snapshots, hashtag sets, and audience notes

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-social-research/sdk`) — Core library for Node.js
- **Python SDK** (`minions-social-research`) — Python bindings
- **CLI Tool** (`@minions-social-research/cli`) — Command-line interface
- **Web App** — Interactive playground at [social-research.minions.wtf](https://social-research.minions.wtf)
